## 📋 الخطوات بالتفصيل

### الخطوة 1: ارفع الكود على GitHub (5 دقايق)

#### أولاً: لو معندكش حساب GitHub
1. روح على https://github.com
2. اضغط "Sign up"
3. اعمل حساب (مجاني)

#### ثانياً: ارفع الكود

افتح Terminal (الطرفية) في مجلد المشروع واكتب:

```bash
# 1. ابدأ git
git init

# 2. ضيف كل الملفات
git add .

# 3. احفظ التغييرات
git commit -m "جاهز للنشر"

# 4. روح على GitHub وعمل مستودع جديد
# اذهب إلى: https://github.com/new
# اسم المستودع: zad-muslim
# اختار Public أو Private (زي ما تحب)
# اضغط "Create repository"

# 5. اربط المستودع وارفع الكود
# غير YOUR_USERNAME باسم المستخدم بتاعك
git remote add origin https://github.com/YOUR_USERNAME/zad-muslim.git
git branch -M main
git push -u origin main
```

**تمام!** الكود دلوقتي على GitHub.

---

### الخطوة 2: انشر على Render (5 دقايق)

#### 1. اعمل حساب على Render
- روح على https://render.com
- اضغط "Get Started for Free"
- اختار "Sign up with GitHub" (أسهل طريقة)
- وافق على ربط Render بـ GitHub

#### 2. اعمل Web Service جديد
- اضغط على زرار "New +" (فوق على اليمين)
- اختار "Web Service"
- اضغط "Connect" جنب مستودع `zad-muslim`
- لو مش لاقيه، اضغط "Configure account" وادي صلاحيات

#### 3. اضبط الإعدادات

**الإعدادات الأساسية:**
```
Name (الاسم): zad-muslim
Region (المنطقة): اختار الأقرب ليك (مثلاً Frankfurt لو في مصر/الخليج)
Branch (الفرع): main
Runtime (بيئة التشغيل): Node
```

**أوامر البناء والتشغيل:**
```
Build Command (أمر البناء): npm install && npm run build
Start Command (أمر التشغيل): npm start
```

**الباقة:**
```
اختار: Free (مجاني)
```

#### 4. ضيف متغيرات البيئة

اضغط "Advanced" → "Add Environment Variable"

ضيف دول واحدة واحدة:

**المتغير الأول:**
```
Key (المفتاح): DATABASE_URL
Value (القيمة): file:./prisma/dev.db
```

**المتغير الثاني:**
```
Key: NEXT_PUBLIC_VAPID_PUBLIC_KEY
Value: BDfKkya975WhlfmZAkyxvBaiInsvJ-RyqNIZXyhPWTIemqB8nfmrhLruP2r1Yizaw915G_qSZG8pMl6-f0xUIM8
```

**المتغير الثالث:**
```
Key: VAPID_PRIVATE_KEY
Value: 6IDB6Vntsb7D9yTa18iCLP8KWsd4CzUL1IBc9pTl66I
```

**المتغير الرابع:**
```
Key: VAPID_SUBJECT
Value: mailto:zad-muslim@app.com
```

**المتغير الخامس:**
```
Key: NODE_ENV
Value: production
```

#### 5. ضيف مساحة تخزين (للداتابيز)

**مهم جداً:** ده علشان قاعدة البيانات تفضل محفوظة!

- انزل تحت لحد قسم "Disks"
- اضغط "Add Disk"
- الإعدادات:
  ```
  Name (الاسم): zad-data
  Mount Path (مسار التثبيت): /app/prisma
  Size (الحجم): 1 GB (مجاني)
  ```
- اضغط "Save"

#### 6. انشر التطبيق!

- اضغط "Create Web Service"
- استنى 5-10 دقايق للنشر الأول
- اتفرج على السجلات (Logs) - هتشوف:
  - تثبيت المكتبات
  - بناء Next.js
  - تشغيل السيرفر

**تمام!** التطبيق بقى شغال على: `https://zad-muslim.onrender.com`

**تمام!** التطبيق هيفضل صاحي 24/7.
